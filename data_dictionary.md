# Data Dictionary

bokning_id – Unique booking identifier  
medlem_id – Unique member identifier  
medlemstyp – Membership type (Basic, Premium, Student)  
medlem_startdatum – Membership start date  
medlem_slutdatum – Membership end date  
månadskostnad – Monthly membership cost (SEK)  
födelseår – Birth year of member  
age – Calculated age  
pass_id – Unique class ID  
passnamn – Class name (Yoga, Pilates, etc.)  
anläggning – Gym location  
instruktör – Instructor name  
status – Booking outcome (Completed, Cancelled, No Show)  
feedback_text – Written customer feedback  
feedback_betyg – Rating from 1–5  
sentiment_raw – Raw sentiment output from BERT  
sentiment – Simplified sentiment category (Positive, Neutral, Negative)
